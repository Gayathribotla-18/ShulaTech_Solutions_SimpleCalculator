Java Calculator App

This is a simple calculator application made using Java Swing.
It performs basic arithmetic operations and also shows calculation history.


✨ Features
Basic arithmetic operations (+, -, *, /, %)
Clear (AC) and Delete (DEL) buttons
Calculation history display
Dark and light theme switch
Simple phone-style UI


🛠️ Tech Used :
Java
Swing (GUI)
AWT


▶️ How to Run
Open CMD in the project folder and run:
javac CalculatorApp.java
java CalculatorApp


📌 Note

This project is made for learning Java GUI (Swing).
No external libraries are used.
